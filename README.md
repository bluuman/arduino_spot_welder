# arduino_spot_welder
In this repository is the code to the arduino spot welder, as seen [on youtube](https://www.youtube.com/watch?v=RaHopv_f08A).

![Circuit](/circuit.png)

## Materials
8x MOSFET PSMN2R0-30PL in Parallel

1x VS-100BGQ045

1x Foot Switch

1x >=600 CCA Battery

2x Electrode (Represented as inductor coils)
